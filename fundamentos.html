🤔 ¿Qué es una Base de Datos?

Una base de datos es una colección organizada de información estructurada que se almacena electrónicamente en un sistema informático. Permite guardar grandes cantidades de datos para que puedan ser consultados, actualizados y gestionados de forma rápida y eficiente.


⚔️ Archivos de Texto Plano vs. SGBD

Para guardar datos podríamos usar un simple archivo `.txt`, pero a gran escala esto genera graves problemas. Veamos la diferencia frente a usar un Sistema de Gestión de Bases de Datos (SGBD) como MySQL:

Característica Archivos de Texto Plano (.txt) SGBD (Ej. MySQL)
Velocidad Lenta. Hay que leer todo el archivo línea por línea para buscar un dato. Ultra rápida gracias al uso de índices avanzados.
Concurrencia Si dos usuarios intentan escribir a la vez, el archivo se corrompe o se bloquea. Permite que miles de usuarios lean y escriban datos en simultáneo de forma segura.
Seguridad Cualquiera con acceso al archivo puede ver o modificar todo el contenido. Posee control de usuarios, permisos específicos y cifrado de datos.
Duplicidad Es fácil repetir información por error (ej. registrar dos veces el mismo correo). Evita la redundancia mediante reglas de integridad automáticas.
